Simple HA integration for Awenta AHR recuperation units, I did not add "hidro" mode or night time, since this is easier to set up in HA by automations.

1.Download manually or through HACS via link.<br>
2.Add integration and enter your email and pass for Awenta AHR app.<br>
3.Done!
